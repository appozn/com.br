# 🚀 COMO USAR - Notificações do Computador para o Celular

## Passo 1: Instalar e Iniciar (PRIMEIRA VEZ)

1. **Clique duas vezes** no arquivo `INICIAR.bat`
2. Aguarde a instalação automática das dependências
3. O servidor iniciará automaticamente

## Passo 2: Acessar no Computador

1. Abra o navegador no **computador**
2. Acesse: `http://localhost:3000`
3. Faça login (admin@bunny.app / 12345)

## Passo 3: Acessar no Celular

1. **O servidor mostrará um endereço** tipo: `http://192.168.1.10:3000`
2. Abra o navegador do **celular**
3. Digite esse endereço
4. Faça login
5. **Clique no sino 🔔** e permita notificações

## Passo 4: TESTAR! 🎉

1. **No celular**: Bloqueie a tela
2. **No computador**: Crie uma venda
3. **MÁGICA**: A notificação aparecerá no celular bloqueado!

## Como Funciona

- ✅ **Banco de Dados SQLite**: Todas as vendas são salvas permanentemente
- ✅ **WebSocket**: Sincronização em tempo real entre dispositivos
- ✅ **Service Worker**: Notificações aparecem mesmo com celular bloqueado

## Requisitos

- Node.js instalado (https://nodejs.org)
- Computador e celular na mesma rede Wi-Fi

## Problemas Comuns

**"Não conecta":**
- Verifique se computador e celular estão na mesma rede
- Desative firewall temporariamente

**"Node.js não encontrado":**
- Instale em: https://nodejs.org
- Reinicie o computador
